Grid formatting

  • To put several elements in a row, use the row class
  • Each element that belongs in a row is of the col class
  • Each row can hold a max of 12 columns
  • You can specify how many columns each element spans by putting an integer at the of the class name, like so: class="col-[number of columns]"
  • Try changing the column width of the text above (line 43-54 in source code). They should get narrower/wider depending on the number of columns given.
  • Try removing -[num of cols] after every col element to spread them out equally across

Media Objects

  • The media class saves you the trouble of trying to make some content appear next to another element
  • Place elements in the media-body and they will show up side-by-side with whatever else is wrapped by the media block
